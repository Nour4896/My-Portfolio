Simulation Hockey League Banking Web Portal

The Simulation Hockey League's Hamilton Steelhawks, under new ownership, were getting ready to move for a new home in Pittsburgh, Pennsylvania. This rebrand aimed to give them a frehs new look in celebration of this new era, complete with new logos and jerseys, created in Adobe Photoshop and Illustrator.

Project completed for Google UX Design Certificate. This redesign aimed to make sweeping improvements to the user experience provided by the Freedom Mobile app, completely revamping it with a modernized look, better presentation, and more organized information, created in Figma.

Project completed for Graphic Design Program at Seneca College. This 12-page brochure details the workflow and creative process of Super Smash Bros Director Masahiro Sakurai, as part of a series of designer brochures for Design Exchange Toronto, created in Adobe InDesign

With the Simulation Hockey League's move away from forum-based account management, the Dev Team worked to create a portal for users to conduct their bank transactions. This project involved interviewing SHL users, creating prototypes, and conducting tests to ensure the new portal provided a great user experience. Created using Figma.
